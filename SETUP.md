# Инструкция по запуску

## Быстрый старт

1. **Установите зависимости:**
```bash
npm install
```

2. **Создайте файл `.env` в корне проекта:**
```env
DATABASE_URL="file:./dev.db"
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="change-this-to-a-random-string-in-production"
ADMIN_EMAIL="admin@example.com"
ADMIN_PASSWORD="admin123"
```

3. **Инициализируйте базу данных:**
```bash
npm run db:push
npm run db:generate
```

4. **Создайте администратора:**
```bash
node scripts/create-admin.js
```

5. **Запустите сервер разработки:**
```bash
npm run dev
```

6. **Откройте браузер:**
- Публичный сайт: http://localhost:3000
- Админ панель: http://localhost:3000/admin/login

## Первые шаги после запуска

1. Войдите в админ панель используя данные из `.env`:
   - Email: `admin@example.com`
   - Пароль: `admin123`

2. Добавьте услуги:
   - Перейдите в "Услуги" → "Добавить услугу"
   - Заполните название, описание, цену и длительность

3. Добавьте примеры работ:
   - Перейдите в "Портфолио" → "Добавить работу"
   - Загрузите изображение и заполните информацию

4. Создайте временные слоты:
   - Перейдите в "Временные слоты" → "Добавить слот"
   - Выберите дату и время

5. Теперь клиенты могут записываться через публичную часть сайта!

## Важные замечания

- Убедитесь, что папка `public/uploads` существует для загрузки изображений
- В production измените `NEXTAUTH_SECRET` на случайную строку
- Для production рекомендуется использовать PostgreSQL вместо SQLite
- Все изображения сохраняются в `public/uploads/`

## Структура базы данных

- **User** - администраторы системы
- **Service** - услуги маникюра
- **PortfolioItem** - примеры работ
- **TimeSlot** - доступные временные слоты
- **Booking** - записи клиентов
